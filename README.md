# JavaScript Short Answers

### 1️⃣ Difference between `var`, `let`, and `const`

* **var**: Function scoped, can be redeclared and updated.
* **let**: Block scoped, can be updated but not redeclared in the same scope.
* **const**: Block scoped, cannot be updated or redeclared.

### 2️⃣ What is the Spread Operator (`...`)?

The spread operator is used to expand elements of an array or object. It helps copy or merge arrays and objects.

### 3️⃣ Difference between `map()`, `filter()`, and `forEach()`

* **map()**: Returns a new array after modifying elements.
* **filter()**: Returns a new array with elements that match a condition.
* **forEach()**: Loops through elements but does not return a new array.

### 4️⃣ What is an Arrow Function?

An arrow function is a shorter syntax for writing functions using `=>`.

### 5️⃣ What are Template Literals?

Template literals are strings written with backticks (`` ` ``) that allow variables inside strings using `${}`.
